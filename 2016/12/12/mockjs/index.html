模拟数据生成器

MockJs

作用

  • Mock.js 是一个模拟数据生成器,可以让我们独立于后端进行开发,基于require.js和jquery.js
  • 作用:
  1. 根据数据模版生成模拟数据
  2. 模拟ajax请求,生成并返回模拟数据
  3. 基于html模版生成模拟数据

语法规范

  1. 数据模版定义
  2. 数据占位符定义

数据模版的定义

  • 每个属性有3个部分构成:属性名/生成规则/属性值
  • 属性名和生成规则之间用”|”隔开
  • 生成规则是可选择的,一共有7种
  1. ‘name|min-max’: value
  2. ‘name|count’: value
  3. ‘name|min-max.dmin-dmax’: value
  4. ‘name|min-max.dcount’: value
  5. ‘name|count.dmin-dmax’: value
  6. ‘name|count.dcount’: value
  7. ‘name|+step’: value
  • 属性值可以是字符串,数组,数字,对象,布尔值,函数或者正则表达式,函数则取其返回值作为最终的属性值

数据占位符的定义

  • 占位符顾名思义就是在属性字符串中占个位置,并不会在最后生成的数据中出现
  • @标识其后面的字符串是占位符
  • 占位符引用的是Mock.Random中的方法
  • 用户可以通过Mock.Random.extend()来拓展自定义占位符
  • 占位符会优先引用数据模版中的属性
  • 占位符支持相对路径和绝对路径

拦截ajax

  • Mock.setup(settings) 配置项只有一个timeout,用于设置指定被拦截的ajax请求的响应时间,单位是毫秒,可以使确切的一个值,也可以是一个区间0-300等;
  • stringify 的参数 第一个参数为转换的数据,第二个参数为转换的数据包含的内容,可以传入需要的keys数组,也可以传入回调函数来处理传输的数据.第三个参数为格式化字符串数据用的.
